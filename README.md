# Solana-Nft-Gallery
Solana NFT Solana

## Installation

Use the package manager [npm](https://pip.pypa.io/en/stable/) to install foobar.

```bash
cd gallery-3d/star-website-master
npm install
npm run build
npm start
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
