### Development

Create the config file for all assets

```sh
$ npm run config
```

Install

```sh
$ npm install
```

Run

```sh
$ npm start
```

Go to
```sh
localhost:8080
```

### Docker

Installer docker-compose [ici][df1]

Build project images

```sh
$ docker-compose build
```

Run project

```sh
$ docker-compose up
```

Go to
```sh
localhost:80
```

### To convert GIF to OGV

Install ffmpeg

```sh
$ apt install ffmpeg
$ ffmpeg -i [PATH_VIDEO.gif] [PATH_VIDEO.ogv]
```
